# 🏦 Projeto ContaBanco

## 📖 Sobre o Projeto

Este projeto foi desenvolvido durante o Bootcamp **Claro - Java com Spring Boot**, promovido pela Digital Innovation One (DIO).

O desafio faz parte da Trilha Java Básico e tem como objetivo praticar os fundamentos da linguagem Java por meio da construção de uma aplicação de terminal que simula a abertura de uma conta bancária.

A implementação foi baseada no desafio oficial proposto pelo instrutor **Gleyson Sampaio**, utilizando entrada de dados via terminal com a classe `Scanner` e exibição dinâmica das informações fornecidas pelo usuário.

A proposta consiste em criar uma aplicação simples de terminal capaz de simular a abertura de uma conta bancária, permitindo ao usuário informar seus dados e receber uma mensagem personalizada de confirmação.

## 🎯 Objetivo do Desafio

Praticar os conceitos fundamentais da linguagem Java, com foco na sintaxe, declaração de variáveis, interação com o usuário e organização básica do código.

## 📋 Funcionalidades

- Solicitar dados da conta bancária via terminal;
- Receber entradas do usuário utilizando `Scanner`;
- Armazenar informações em variáveis;
- Exibir mensagem personalizada de boas-vindas;
- Simular a criação de uma conta bancária.

## 🛠️ Conceitos Aplicados

Durante o desenvolvimento deste projeto foram praticados os seguintes conceitos:

- Anatomia das classes;
- Tipos e variáveis;
- Operadores;
- Métodos;
- Escopo;
- Palavras reservadas;
- Entrada de dados pelo terminal (`Scanner`);
- Concatenação de Strings utilizando o método `concat()`.
